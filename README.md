## 🚀 Sobre o Projeto

Este projeto é o meu cartão de visitas digital. Ele foi desenvolvido para consolidar minha trajetória como estudante de **Análise e Desenvolvimento de Sistemas (FATEC)** e Técnico em **Automação Industrial (ETEC)**. 

Mais do que uma página estática, este portfólio apresenta:
- **Timeline de Projetos:** Uma jornada interativa pelos meus principais desenvolvimentos (Python, Java, IoT).
- **Galeria de Certificados:** Carrossel dinâmico exibindo minhas conquistas acadêmicas.
- **Modo Dark/Light:** Interface adaptável à preferência do usuário.
- **Responsividade Total:** Experiência otimizada para Mobile e Desktop.
